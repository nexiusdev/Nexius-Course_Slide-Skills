# Sourcing Notes

These sources are useful references for PowerPoint illustration work. Always
check current licensing and access before downloading or embedding assets.

## Slidesgo

Use for illustrated presentation style references, PowerPoint-ready templates,
and format/style filters such as 16:9, dark, professional, modern, 3D,
isometric, and infographics. Free and premium templates may have different
download limits or licensing requirements.

## IconScout

Use for illustration packs and individual assets. Relevant formats include PNG,
SVG, EPS, AI, and JPG. Check whether an item is free or premium and whether the
license permits the intended presentation use before embedding it.

## Envato Elements

Use as a paid template/reference marketplace. Do not download or use assets
unless the user has a valid license/subscription and explicitly asks to use that
licensed asset.

## SlideModel

Use as a reference for editable PowerPoint diagrams, infographic templates,
visual metaphors, process diagrams, and illustration templates. It is especially
useful for deciding whether a slide needs an infographic, standalone image, or
editable PowerPoint illustration.
